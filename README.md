### Install
```
npm install --save vuc-input || vuc-carousel ||.....

yarn add vuc-input || vuc-carousel || ......
```

### Basic use
```
// vuc-input package
// Global

import vucInput from 'vuc-input'
Vue.use(vucInput)

// Local
import {vucInput} from 'vuc-input'
export default {
  components: {vuc-input}
}
```
